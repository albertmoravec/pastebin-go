About

Simple pastebin written in GOLANG.

API

Make a POST request like this:

curl -F"p=$(xsel)" {{ .AppUrl }}

You can set when the paste expires. The following paste will expire in 60 seconds:

curl -F"p=$(xsel)" -F"expire=60" {{ .AppUrl }}
